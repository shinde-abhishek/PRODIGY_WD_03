# ❌⭕ Tic-Tac-Toe Game

A classic 2-player Tic-Tac-Toe game built with vanilla JavaScript, HTML, and CSS. Play X and O turn-by-turn and enjoy win/draw logic.

## 🎯 Features
- Two-player mode
- Detects win, draw, and invalid moves
- Simple and clean UI
- Reset game button

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
